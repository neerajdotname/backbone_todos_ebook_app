Explanation

To display the list of todo items, in the initialize method fetch method is being invoked. When collection invokes fetch then usually data from server is received. However in this case data from backbone-localstorage will be fetched. Once the data arrives then reset callback is invoked. This commit makes use of reset callback to display data.
