Introduction

backbone.js brings MVC style of development to JavaScript. The source code of backbone.js has todos application which can be seen in action here: http://documentcloud.github.com/backbone/examples/todos/index.html

Though the source code of todos is simple, it might seem intimidating to some. This ebook is my attempt to build the application in 18 incremental steps. The first step will not have any JavaScript code but slowly JavaScript code will be added.

Backbone.js is undergoing a lot of change very rapidly. As of this writing Backbone.js 0.5.2 was the most stable release and thatâ€™s what is used in this application.
